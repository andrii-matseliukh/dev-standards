# dev-standards

My default developer configs — editor, formatting, Claude Code, and extensions — ready to drop into any project.

## What's inside

| Path | Purpose |
| --- | --- |
| `.editorconfig` | C# / .NET code style: naming, `var` usage, expression bodies, using sorting (Rider-parity). |
| `.csharpierignore` | Files CSharpier skips when formatting. |
| `.vscode/settings.json` | Editor look & feel: Catppuccin Mocha theme, Roboto Mono, inlay hints, format-on-save off. |
| `.vscode/extensions.json` | Recommended extensions (frontend, .NET, Azure, databases, AI). |
| `.vscode/launch.json` | Debug profiles for single-file and ASP.NET `dotnet run`. |
| `.claude/settings.json` | Claude Code hooks and permission rules. |
| `.claude/hooks/` | Scripts: notification sound, context-usage warning, command logging. |
